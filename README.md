# dotkms

A handy tool for KMS (Key Management System). There is only support for Google Cloud KMS.

## Requirements

* [gcloud](https://cloud.google.com/sdk/gcloud)

## Installing

```bash
$ npm install dotkms
```

or 

```bash
$ yarn add dotkms
```

## Usage

Please see the help

```bash
$ dotkms --help
$ dotkms encrypt --help
$ dotkms decrypt --help
```
